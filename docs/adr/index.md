# Architecture Decision Records

This section documents the architectural and technical decisions made in Net Worth Tracker.

ADRs capture context, reasoning, and trade-offs at the time a decision was made. They are not intended to be revised retroactively — if a decision changes, a new ADR supersedes the old one.
